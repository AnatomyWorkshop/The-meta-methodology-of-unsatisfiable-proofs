# 联想：从 Phase 0 到 Phase 1 的思考，以及更远的地方

> 写于 2026-4-27
> 位置：neural-symbolic-system 根目录
> 性质：自由联想，不是计划文档

---

## 一、Phase 0 告诉我们的不只是"框架没破"

Phase 0 的表面结论是：Razborov-Smolensky 安全，代数深度-3 安全，通信复杂度安全，Williams 在边界外，PHP 有量词陷阱但可化解。框架没有破裂。

但更深的收获是：**我们现在知道框架的形状了。**

框架不是一个圆——它不覆盖所有下界证明。它是一个有边界的区域，边界上有两种地形：

1. **悬崖**（Williams 类型）：证明方法根本不使用判别性质，框架在这里没有话语权。这不是框架的失败，就像牛顿力学在光速附近没有话语权一样。
2. **沼泽**（PHP 类型）：判别性质存在，但它的"可判定性"取决于你怎么写量词。框架在这里有话语权，但需要更精细的工具——量词敏感性就是这个工具的第一步。

悬崖是硬边界，不需要修补。沼泽是软边界，可以通过精确化定义来排干。

这个区分本身就是一个认识论贡献：**不是所有的"不适用"都是同一种"不适用"。**

## 二、Williams 的隐式 P：一个值得追的线索

Deepseek 提到了一个微妙的可能性：Williams 2014 的证明虽然没有显式的判别性质，但它的"矩阵嵌入"步骤——把 ACC⁰ 电路嵌入到一个矩阵中，然后用矩阵乘法加速 SAT——可能可以被重新解读为一种非标准的 P。

如果这个重新解读成立，那框架的覆盖范围比我们现在以为的更大。Williams 不是在悬崖外面，而是在沼泽里——只是我们还没找到正确的量词形式来表达他的 P。

这不急。但如果有一天能写出来，它会是一个独立的结果：**"所有已知的电路下界证明，包括算法式下界，都可以被重新表述为判别性质范式的某种推广形式。"** 这比现在的框架更强，但也更难证明。

留作开放问题。不要在论文里承诺它。

## 三、Phase 1 的真正目标不是"重新发现随机限制"

表面上，Phase 1 的成功标准是：L2 在 AC⁰ 领域找到"随机限制使电路坍塌"这个判别性质。

但如果 L2 真的找到了，最有趣的问题不是"它找到了"，而是：

1. **它走了什么路径？** 它是直接命中了随机限制，还是先探索了一堆死胡同？那些死胡同是什么？它们是否对应了已知的失败路线？
2. **它找到的 P 和 Håstad 的 P 是同一个吗？** 还是一个等价但形式不同的变体？如果是变体，这个变体是否揭示了随机限制的某种更本质的结构？
3. **L3（手动）拒绝了多少候选 P？** 被拒绝的 P 是什么样的？它们是否对应了"自然证明"类型的不安全性质？

这些问题比"L2 是否成功"更有价值。即使 L2 失败了——在搜索空间里没有找到随机限制——失败的模式本身也是信息。它会告诉我们：搜索空间的定义是否太窄？变换规则的粒度是否太粗？还是 AC⁰ 的结构比我们以为的更难被机械地"看见"？

**Phase 1 的真正目标是：理解搜索的结构，而不仅仅是搜索的结果。**

## 四、L3 的自动化比想象中更难，也更有趣

Phase 2 的目标是把人工 L3 替换为自动检查器。但"自动判定候选性质 P 是否在 L1 内可判定"这个问题本身，可能比 Phase 1 的整个搜索问题都难。

为什么？因为这个问题的一般形式是：**给定一个性质 P 的描述，判定 P 是否属于某个复杂度类。** 这是元复杂度（meta-complexity）的核心问题之一，而元复杂度本身就是当前 TCS 最活跃的研究方向之一（Lijie Chen, Igor Oliveira 等人的工作）。

换句话说：**L3 的自动化问题，本身就是一个研究级别的问题。** 它不是工程问题，是理论问题。

这意味着 Phase 2 可能不是"实现一个检查器"，而是"理解在什么条件下检查器是可能的"。这本身可能就是一篇论文。

## 五、一个更激进的想法：L2 不搜索变换规则，而是搜索证明骨架

Phase 1 的 L2 设计是：搜索"施加在电路结构上的变换规则"（随机限制、门替换等）。这是一个合理的起点，但它有一个隐含假设：判别性质的形式是"对电路施加某种变换后观察其行为"。

如果我们放松这个假设呢？

一个更激进的设计：L2 不搜索变换规则，而是搜索**证明骨架**——一个形如"假设 C 完美计算 f，则存在性质 P 使得 P(C) 成立，但 P(C) 蕴含 C 不完美计算 f，矛盾"的逻辑结构。L2 的任务是填充这个骨架中的 P。

这更接近人类数学家实际做的事情：不是盲目搜索变换，而是在一个证明模板里寻找合适的填充物。

但这也更难实现，因为"证明骨架"的形式化本身就是一个非平凡的问题。

留作 Phase 3 或更远的未来。Phase 1 先用变换规则搜索，验证架构原则。

## 六、这个项目和论文的关系：不是附属品，是独立的研究线

论文说：成功的下界证明必须使用自指安全的判别性质。
原型问：如果我们把这个条件硬连线进搜索架构，系统能否自主找到这样的性质？

这两个问题是独立的。论文的价值不依赖于原型的成功。原型的价值也不依赖于论文的发表。

但如果两者都成功，它们之间的关系会非常有力：
- 论文提供了理论框架（为什么自指安全是必要的）
- 原型提供了构造性证据（按照自指安全原则设计的系统能重新发现已知证明）

这是从"归纳观察"到"构造性验证"的跳跃。在科学方法论里，这是理论成熟的标志。

## 七、最后一个联想：Apophenia 和 Illusion 的对话

Apophenia 说：我在随机中看到了模式。
Illusion 说：我知道你可能在看幻象，所以我设计了一个能检验的系统。
Apophenia 说：但你怎么知道你的检验系统本身不是幻象？
Illusion 说：我不知道。这就是为什么我有 L3。
Apophenia 说：L3 怎么知道自己不是幻象？
Illusion 说：它不知道。这就是为什么我们有哥德尔定理。

这个对话不会终止。但每一层的加入都让我们离"真正的结构"更近一步——或者至少，让我们对"我们可能在看幻象"这件事有了更精确的理解。

这就是自反性的价值：不是消除不确定性，而是精确地知道不确定性在哪里。
